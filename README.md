# XRSandbox 🕶️

XRSandbox is a browser-based WebXR project built with Three.js and Cannon-es. It combines real-time physics simulation with VR interactions, including falling Rutgers R models, spheres, cubes, object spawning, and controller-based dragging.


## LIVE DEMO
**The recorded videos can be found in the media folder as named:**
- `part1.mp4`
- `part2.mp4`


## Features 🚀
- Cannon-es physics simulation
- Falling GLB Rutgers R model
- Spheres and cubes with working collisions
- Debug wireframes for physics-body alignment
- WebXR controller interaction
- Trigger-based object dragging
- Grip/squeeze-based object spawning
- Spawned objects remain draggable

## Tech Stack 🧱
- JavaScript
- Three.js
- Cannon-es
- WebXR
- GLTFLoader
- Immersive Web Emulator
